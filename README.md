# Generador de Rutinas

Este proyecto es un simulador de rutinas de gimnasio desarrollado en JavaScript como parte del curso de Coderhouse.

## Funcionalidad

El programa solicita al usuario:

- Nombre
- Cantidad de días de entrenamiento por semana

En base a esa información, genera una rutina personalizada mostrando los días y grupos musculares en la consola.

## Tecnologías utilizadas

- HTML
- JavaScript
